PacePal: A Geo-Aware Retrieval-Augmented Chatbot Agent for Walking Engagement and Well-Being

PacePal is a geo-aware retrieval-augmented conversational agent designed to provide grounded, location-relevant walking support while maintaining clear safety boundaries and a well-being-oriented interaction style.

Project pageLatest news

Plain-language summary

Walking-support chatbots need to understand where a user is, retrieve relevant evidence, and avoid unsupported advice. PacePal combines geographic context with retrieval-augmented generation so that responses are grounded in relevant information rather than generated from model memory alone.

Research contribution

  • Geo-aware retrieval for context-sensitive walking-support dialogue.
  • Dense retrieval, reranking, contextual compression, and response generation in one pipeline.
  • Safety guardrails and system-level evaluation of response quality and latency.
